# KNft
这是一个Jfinal框架的课程设计

主题内容是设计了数字藏品的购买平台，分为购买平台的前台和管理员的管理后台

前端运用了JQuery，bootstrap框架，后端用了Jfinal，用了enjoy模板和Ajax异步等方式交互数据，实现了购买，充值，实时点赞功能，管理藏品分页等功能，具体见word文档。
